八、CSS Tricks Yin Yang 笔记

###原文链接
首先要设置上下颜色不一样的两个长方形：

.yin-yang{
    width: 96px;
    height: 48px;
    background: #eee;
    border-color: red;
    border-style: solid;
    border-width: 2px 2px 50px 2px;
}
宽度是高度的一半，将长方形的背景色设置为background: #eee;下半部分的颜色是直接将border-bottom设置成50px,视觉上就又了两个高度和宽度都一样的长方形（因为50px的下边框-2px的边框粗细=48px）。如图：
接着将图形变成上下两个颜色的圆形，添加以下： 
.yin-yang{
    .  .  .  .  .  .
    border-radius: 50%;
}
如下图所示:
利用伪元素画图中的中心镂空的圆形： 
.yin-yang:before {
    content: "";
    position: absolute;
    top: 50%;
    left: 0;
    background: #eee;
    border: 18px solid red;
    border-radius: 50%;
    width: 12px;
    height: 12px;
}
.yin-yang{
    . . . . . .
    position: relative;
}
利用绝对定位将伪元素定位到距离左边0px（left:0px）,距离顶部50%(top:50%)的地方，.yinyang视觉上的高度超过48px，但是那是包括border-bottom的高度；这里50%要注意是相对.yinyang48px的高度而言的。 要将两个伪元素圆形左右正好平均分布在父元素宽度上，就要使每个伪元素圆形的宽度为48px(不用减去边框的宽度，因为left:0px;),可以看出，18*2+12=48； 需要将伪元素的高度设为此时是相对于父元素的，所以.yiyang设置为相对定位(position: relative;)。如图：
接下来用同样的方法将右边的伪元素圆形也放上去: 
.yin-yang:after {
    content: "";
    position: absolute;
    top: 50%;
    left: 0;
    background: red;
    border: 18px solid #eee;
    border-radius:50%;
    width: 12px;
    height: 12px;
}
如图所示:
完整代码如下: 
.yin-yang {
    width: 96px;
    height: 48px;
    background: #eee;
    border-color: red;
    border-style: solid;
    border-width: 2px 2px 50px 2px;
    border-radius: 50%;
    position: relative;
}
.yin-yang:before {
    content: "";
    position: absolute;
    top: 50%;
    left: 0;
    background: #eee;
    border: 18px solid red;
    border-radius: 50%;
    width: 12px;
    height: 12px;
}

.yin-yang:after {
    content: "";
    position: absolute;
    top: 50%;
    right: 0;
    background: red;
    border: 18px solid #eee;
    border-radius:100%;
    width: 12px;
    height: 12px;
}
