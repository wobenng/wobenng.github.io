HTML 元素

1.空元素:没有内容的元素,没有结束标签.

2.替换元素:元素可替换内容部分并非由文档内容直接表示.如<img>标签的图片就是由文档之外提供的

3.<iframe>:将其他Web文档嵌入到当前文档中;它有以下属性:

  • frameborder:绘制边框,值为0则没有边框.

  • src:嵌入路径

  • width和height规定<iframe>的宽度和高度.
